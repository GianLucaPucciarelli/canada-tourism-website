# canada-tourism-website

## 📌 Descripción
Sitio web de turismo sobre Canadá desarrollado con HTML y CSS.  
Presenta información sobre destinos, cultura y atractivos turísticos del país.

## 🎯 Objetivo
Practicar maquetación web, diseño responsive y estructura de páginas.

## 🛠️ Tecnologías
- HTML5
- CSS3

## 💡 Características
- Diseño moderno y limpio
- Estructura organizada
- Responsive

## 🚀 Cómo usar
1. Descargar el repositorio
2. Abrir `index.html` en el navegador
